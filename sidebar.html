📖 使用說明

🖱️ 基本操作

  • 點擊分頁 → 切換到該分頁
  • 點擊 × 按鈕 → 關閉分頁
  • 點擊 + 按鈕 → 新增分頁
  • 拖曳分頁 → 重新排序
  • 右鍵點擊 → 開啟功能選單

🔍 搜尋分頁

  • 在搜尋框輸入關鍵字
  • 可搜尋分頁標題或網址
  • 按 Ctrl+F 快速聚焦搜尋框
  • 按 Esc 清除搜尋

📌 固定分頁

  • 右鍵點擊分頁 → 選擇「固定/取消固定」
  • 固定的分頁會顯示在最上方
  • 固定分頁不會被「關閉其他分頁」關閉

📁 分頁群組

  • 右鍵點擊分頁 → 選擇「建立群組」
  • 點擊群組標題可展開/收合
  • 右鍵 → 「移出群組」可將分頁移出

🎨 主題設定

  • 點擊 ⚙ 設定按鈕
  • 選擇喜歡的主題顏色
  • 設定會自動儲存

🖥️ 全螢幕模式

  • 點擊工具列的 全螢幕按鈕 進入全螢幕
  • 再次點擊可退出全螢幕模式
  • 按 F11 也可以切換全螢幕

⚡ 快捷網站

  • 底部「快捷網站」區塊可快速開啟常用網站
  • 點擊 + 按鈕新增快捷網站
  • 可選擇「加入目前分頁」或手動輸入
  • 滑鼠移到快捷網站上，點擊 × 可刪除

📍 側邊欄位置

  • 在 ⚙ 設定中查看詳細說明
  • 需到 Chrome 設定 → 外觀 → 側邊欄 更改

⌨️ 快捷鍵

Ctrl+F聚焦搜尋框
Esc清除搜尋 / 關閉選單

⚙ 設定

側邊欄位置需在瀏覽器設定中更改：

  1. 點擊 Chrome 右上角 ⋮ → 設定
  2. 左側選單點擊 外觀
  3. 找到 側邊欄 選項
  4. 選擇「顯示在左側」或「顯示在右側」
