## 📥 安裝方式

### 步驟 1：下載專案

```
點擊綠色「Code」按鈕 → Download ZIP → 解壓縮
```

### 步驟 2：生成圖示

```
用瀏覽器開啟 generate-icons.html → 下載圖示到 icons 資料夾
```

### 步驟 3：載入擴充功能

```
chrome://extensions/ → 開啟「開發人員模式」→ 載入未封裝項目 → 選擇專案資料夾
```

### 步驟 4：開始使用

```
點擊工具列擴充功能圖示 → 開啟側邊欄
```

## 🖱️ 操作說明

<table>
<tr>
<td width="50%">

### 基本操作
| 操作 | 說明 |
|:---:|:---|
| 點擊分頁 | 切換到該分頁 |
| 點擊 × | 關閉分頁 |
| 點擊 + | 新增分頁 |
| 拖曳分頁 | 重新排序 |
| 右鍵點擊 | 開啟選單 |

</td>
<td width="50%">

### 快捷鍵
| 快捷鍵 | 功能 |
|:---:|:---|
| `Ctrl+F` | 聚焦搜尋框 |
| `Esc` | 清除搜尋/關閉選單 |

### 工具列按鈕
| 按鈕 | 功能 |
|:---:|:---|
| ⚙ | 設定（主題顏色） |
| ⛶ | 全螢幕切換 |
| ? 說明 | 內建使用教學 |

</td>

## ⚡ 快捷網站

<div align="center">

**底部「快捷網站」區塊可快速開啟常用網站**

</div>

```
點擊 + 按鈕 → 選擇「加入目前分頁」或手動輸入 → 完成！
```

> 💡 預設已包含 YouTube 快捷按鈕

## 📍 側邊欄位置設定

<div align="center">

**側邊欄位置需在瀏覽器設定中更改**

</div>

```
Chrome 設定 → 外觀 → 側邊面板位置 → 選擇「顯示在左側」或「顯示在右側」
```
